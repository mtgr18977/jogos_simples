Ajuda - Tetris

Objetivo do Jogo

O objetivo do Tetris é encaixar as peças (Tetrominos) que caem no campo de jogo de forma que completem linhas horizontais sem espaços vazios. Cada linha completa é removida, e o jogador ganha pontos. O jogo termina quando as peças alcançam o topo do campo.

Controles

  • ← Mover Peça para a Esquerda
  • → Mover Peça para a Direita
  • ↓ Acelerar a Queda da Peça
  • ↑ Rotacionar a Peça no Sentido Horário
  • Q Rotacionar a Peça no Sentido Anti-Horário
  • Espaço Pausar/Jogar

Como Jogar

- As peças caem automaticamente do topo do campo. Use as setas do teclado para mover as peças para a esquerda ou direita e a seta para baixo para acelerar a queda.

- Pressione a seta para cima ou a tecla `↑` para rotacionar a peça no sentido horário e a tecla `Q` para rotacioná-la no sentido anti-horário.

- Complete linhas horizontais sem espaços vazios para removê-las do campo e ganhar pontos.

- O jogo aumenta de velocidade à medida que você avança nos níveis, tornando as peças caindo mais rápido.

- O jogo termina quando uma nova peça não pode ser colocada no campo sem colidir com as peças existentes.

Dicas

- **Planejamento Estratégico:** Observe as próximas peças para planejar onde encaixá-las melhor.

- **Mantenha o Campo Limpo:** Evite acumular peças em um lado do campo; tente manter uma superfície plana.

- **Rotacione com Sabedoria:** Use as rotações para encaixar peças em espaços apertados ou para completar múltiplas linhas de uma vez.

- **Controle a Velocidade:** Use a aceleração para rapidamente limpar linhas durante momentos críticos.
